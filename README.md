# Checkyourphoto V1.0

Futuristic Forensic Metadata Analysis Tool.

## Features
- Photo & PDF metadata extraction
- AI-powered forensic summaries
- Interactive GPS Map
- Chronological Timeline
- Lead capture system

## Deployment (Render.com)

1. **Connect Repository**: Create a new Web Service on Render and connect this GitHub repo.
2. **Settings**:
   - **Environment**: `Node`
   - **Build Command**: `npm install`
   - **Start Command**: `node index.js`
3. **Environment Variables** (Optional):
   - `OPENAI_API_KEY`: Your OpenAI key for AI summaries.
   - `PORT`: Defaults to 3000.

## Development
```bash
npm install
npm start
```
