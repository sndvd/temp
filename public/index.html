Core Forensic Indicators
🔎 Metadata Stripping

The process of removing all hidden information from a file (GPS, camera model, timestamps). Like a letter with the return address scrubbed off.

🔢 Quantization Tables

Every camera brand has its own "recipe" for saving photos. If a photo says it's from an iPhone but uses a Photoshop recipe, we know it's edited.

🖼️ Thumbnail Mismatch

A discrepancy between the main image and its internal preview. The thumbnail often reveals what was originally in the frame before edits.

📍 GPS/Timezone Inconsistency

When location data doesn't match the reported time. Critical for verifying alibis or event locations.

Common Tampering Patterns
📱 Social Media Sanitization

Metadata is missing or says "WhatsApp/Facebook". Verdict: Original source trail destroyed by the platform.

📸 Screenshot of a Fake

Resolution matches a phone screen. The metadata tells us when the screenshot was taken, not when the content was created.

🖌️ Professional Edit

Software tag mentions Photoshop/GIMP. Explicit proof of software manipulation.

🤖 AI Generation Signature

CreatorTool mentions Midjourney/DALL-E. This image was generated or enhanced by AI, not a raw capture.

Pro Agent Tips
  • Check the Timeline: Compare Capture Date to Modification Date. Gaps > few mins suggest processing.
  • Trust the GPS: Harder to fake than dates. Always verify coordinates on a map.
  • Look for "WhatsApp": If present, the evidence trail is likely already broken.
